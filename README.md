# Number Guessing Game

A simple web-based **Number Guessing Game** built using HTML, CSS, and JavaScript.
The player tries to guess a randomly generated number between **1 and 100** within a limited number of attempts.

---

## 🎮 How the Game Works

1. The system generates a **random number between 1 and 100**.
2. The player enters a guess in the input field.
3. The game checks the guess and gives feedback:

   * **Number is too high**
   * **Number is too low**
   * **You guessed correctly**
4. The player has **10 attempts** to guess the correct number.
5. If the correct number is guessed, the player **wins**.
6. If all attempts are used, the game **reveals the correct number**.

---

## 🚀 Features

* Random number generation
* Input validation
* Displays previous guesses
* Shows remaining attempts
* Hint system (High / Low)
* Game over and restart functionality

## 🧠 Game Logic

* A random number is generated using JavaScript.
* User input is validated to ensure it is between **1 and 100**.
* Each guess is stored and displayed.
* The game provides feedback until:

  * The player guesses correctly, or
  * The maximum number of attempts is reached.
